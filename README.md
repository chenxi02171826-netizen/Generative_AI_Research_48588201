# Generative AI Research Project

## Project Overview
This project explores the use of generative AI tools and evaluates their potential benefits and risks.

## Folder Structure
- 01_Literature_Review: research articles and references  
- 02_Quantitative_Analysis: survey data and analysis files  
- 03_Qualitative_Analysis: interview data and transcripts  
- 04_Drafts_and_Reports: research drafts and final reports  
- 05_Additional_Materials: images and supporting materials  
- 06_Scripts: analysis scripts  

## Data Management
All files are organised using a clear folder structure and consistent naming conventions to improve accessibility and efficiency.

## Access Control
Sensitive data such as interview transcripts and survey responses should be restricted to authorised users.

## Collaboration
Use clear file names and meaningful commit messages when making updates.
